About me

Hi there, my name is Hristina, I'm 27 and I love cats! :3

My origins are from North Macedonia, but for the last 4 years I've made a new home in Germany, in the cozy city of Freiburg. I'm a curious person by nature and love to learn about varieties of topics, but in particular I developed a passion for computers and how they work. You can find me frequently spending my nights tinkering with my linux system, changing keybindings of my tiling window manager (i3) or obsessively designing the polybar to sit juuuust right.

As I dwelled deeper, I've decided to work towards the goal of transforming that passion into a profession, with the hope that I will find opportunities which will allow me to learn more and use that knowledge practically.

Development

Don't let that title fool you!

I'm still fairly new at programming. But, I've managed to do few things:

  • build a website from scratch with HTML, CSS and JS using online tutorials. If you want to check out the code, just head to the buttons down below, where you can check the code as an image, or the github repository.
  • set up a virtual private server using ssh and nginx. The server supports HTTPS thanks to the kind people from LetsEncrypt and is setup to renew the certificate automatically, using a cronjob that periodically runs the certbot.
  • set up a GitHub repository for the website and learned the basics of Git.
  • started using Linux with a focus on the command line. got familiar with GRUB2, started using the i3wm with specific keybindings, also trying to avoid using the mouse (semi-successful try).

With regards to the operating system on the old laptop I was using, changing to Linux was an awesome experience, since that laptop had a CPU speed of [....] and running Windows on it made it extremely slow.

As a beginner, I started with Ubuntu, but still didn't quite liked it, due to the many unnecessary pre-installed programs.

Researching through the many open-source OS, I landed on Debian which offered higher security and had a great manual on it's website of how to set it up yourself.

But, that still hasn't satisfied me fully, because again, it came with pre-installed programs that I didn't needed.

Soon after, Arch Linux came into existence on my computer. It is highly configurable, with only the minimum required programs, has a great community of people who devote their time to maintain it and security-wise is up-to-date.
